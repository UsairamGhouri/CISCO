Cisco Networking Labs
=====================

This repository contains a collection of Cisco Packet Tracer (.pkt) files designed for learning and practicing various networking concepts. These labs cover a range of topics from basic routing to advanced network configurations.

* * * * *

Projects and Assessments
------------------------

-   **Final Project.pkt**: A comprehensive network design likely incorporating multiple advanced networking technologies.

-   **Project.pkt**: A standalone networking project demonstrating core configuration skills.

-   **Assessment-2.pkt**: A practical assessment file used to evaluate networking proficiency.

-   **Assignment-1.pkt**: Initial coursework assignment focused on fundamental networking tasks.

Networking Concepts
-------------------

The following files demonstrate specific protocols and configurations:

-   **VLAN.pkt**: Demonstrates Virtual Local Area Network (VLAN) configuration and inter-VLAN routing.

-   **Static Routing.pkt**: Focuses on the manual configuration of network routes between routers.

-   **VLSM.pkt**: Implementation of Variable Length Subnet Masking (VLSM) for efficient IP address allocation.

-   **FTP x HTTP.pkt**: A lab showcasing the configuration and interaction of File Transfer Protocol (FTP) and Hypertext Transfer Protocol (HTTP) services.

-   **WAN And Topologies.pkt**: Exploration of Wide Area Network (WAN) technologies and various physical/logical network topologies.

-   **Topology 1.pkt**: A baseline network topology used for general practice or as a starting point for more complex builds.

Requirements
------------

-   **Cisco Packet Tracer**: You must have Cisco Packet Tracer installed to open and interact with the .pkt files.
